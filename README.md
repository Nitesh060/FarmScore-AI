# 🌾 FarmScore AI

FarmScore AI is an AI-powered agricultural land assessment platform.

## Features

- Farm Suitability Score
- Satellite-based Analysis
- NDVI & Crop Health
- Rainfall Analysis
- Soil Assessment
- Groundwater Analysis
- Road & Market Accessibility
- Risk Classification
- Professional PDF Report

## Technology

- React
- Node.js
- Google Earth Engine
- Open-Meteo
- SoilGrids
- OpenStreetMap
- Render
